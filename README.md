# AWS-Infrastructure-Automation-and-Deployment
This repository contains code, scripts, and configuration templates for automating AWS infrastructure provisioning, deployment, and monitoring. It demonstrates best practices for Infrastructure as Code (IaC) using tools like AWS CloudFormation, Terraform, and the AWS CLI.


Key features include:

Automated provisioning of compute, storage, and networking resources.
CI/CD pipeline integration using GitHub Actions or AWS CodePipeline.
Security and compliance configurations following AWS Well-Architected Framework.
Logging, monitoring, and alerting setup with CloudWatch and AWS Config.
Example use cases for Lambda functions, API Gateway, ECS/EKS, and S3 hosting.
